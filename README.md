## 💻 Getting Started <a name="getting-started"></a>

> Describe how a new developer could make use of your project.
To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- A browser that supports html5
- Git 
- A GitHub account

### Setup

Clone this repository to your desired folder:

- Go to this repo and copy the project link
        https://github.com/Gdiazdiaz/test-rep
- Open your terminal and clone the repo with this command "git clone https://github.com/Gdiazdiaz/test-rep.git"
- Now, you have a copy on your pc. 


### Install

Install this project with:

- You need to install npm to use the commands
- In your console type, `npm install` to install all the dependencies
- Then use `npm start` to deploy the project in local host

### Usage

To run the project, execute the following command:

npm start

### Run tests

To run tests, run the following command:
npm test

### Deployment

You can deploy this project using:

Github Pages

This project use Github Pages to deploy the page, just type the command: "npm deploy"
